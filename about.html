About

What do you do for recreation? Oh, the usual. I bowl. Drive around. The occasional acid flashback.

Jeff Lebowski, "The Dude"

While my recreational activities aren't nearly as exciting, I enjoy spending my time tinkering with code, administering high performance GNU/Linux systems, and learning how to hack the human genome. I currently work at the Center for Computational Research (CCR) at the University at Buffalo where I split my time between programming and system administration. I like to do many things and tend to keep my plate extremely full.

Software Engineer

As a software engineer at CCR I help support research groups from a wide variety of disciplines utilizing CCR's high-performance computing infrastructure. Many of the projects I support involve writing software to process and analyze large amounts of data. I also provide computational support for UB's Next-Generation Sequencing and Expression Analysis Core, which houses an Illumina HiSeq 2000.

Previously, I worked as a Software Engineer for O'Reilly Media developing web applications and data visualization tools. I spent a considerable amount of time hacking on O'Reilly's corpus of digital content using XQuery, XSLT, and DocBook.

A while back I contributed code to Mark Logic's developer network and wrote cqsh an XQuery command line shell and mlxcc a C API enabling access to Mark Logic from Perl, Ruby, PHP, and Python. I was invited to speak at the 2nd (2005) and 3rd (2006) annual Mark Logic User conferences (before it was Mark Logic World) in which I co-presented cqsh, mlxcc, and projects from O'Reilly Labs.

A long time ago I contributed some Perl code to Word Hacks, despite my loathing for all software originating from Redmond.

You can also find me on GitHub.

System Administrator

As a system administrator at CCR, I write code to automate software installations across thousands of Linux compute nodes using cobbler and puppet. I administer user account management systems using Kerberos and LDAP and maintain core network services such as DNS, DHCP, and NTP.

I build system monitoring tools for tracking resource utilization and energy consumption on high performance compute systems. In addition, I administer web application environments running Apache, Tomcat, and MySQL, and virtual machine infrastructures using KVM and libvirt.

Web Developer

I'm a full stack web developer, i.e., give me Vim and a Linux box and I'll cook up a full blown web application faster that you can finish a ziggy pig.

I lead the development of UBMoD, an open source data warehouse and web portal for mining resource utilization from TORQUE.

I built the web infrastructure for miRdSNP, a database exploring disease-associated SNPs and microRNA target sites on 3' untranslated regions of human genes.

While working at O'Reilly Media, I created a web front end and visualization tools using Treemaps for O'Reilly's research data mart, which includes data on book sales and technology trends. I also was a lead developer for SafariU, a print-on-demand publishing platform that enabled college professors to assemble custom text books using O'Reilly's book chapters and article content.

Bioinformatician in Training

In February of 2013, I completed my masters degree in Bioinformatics and Biostatistics from the University at Buffalo and Roswell Park Cancer Institute. My research focused on algorithms for detecting fusion genes from next-generation sequencing data. Part of my masters thesis was devoted to building a tool called FUSIM, which allows for the complex simulation of fusion transcripts and can be used in testing fusion discovery methods.

A while back I had the opportunity to work on a project which explored the need for biostatisticians to perform correlation analysis on data from array comparative genome hybridization (aCGH). The paper focused on the data-intensive computational problem of processing large correlation matrices on the order of 1 million rows by 1 million columns. I implemented an approach using Hadoop which scaled to 128 compute nodes and queried the top 200 values in under 40 minutes.
