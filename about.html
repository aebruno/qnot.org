About

What do you do for recreation? Oh, the usual. I bowl. Drive around. The occasional acid flashback.

Jeff Lebowski, "The Dude"

While my recreational activities aren't nearly as exciting, I enjoy spending my time tinkering with code, administering high performance GNU/Linux systems, and learning how to hack the human genome. I currently work at the Center for Computational Research (CCR) at the Univeristy at Buffalo where I work as a software/systems engineer. I like to do many things and tend to keep my plate extremely full.

Software Engineer

As a software engineer at CCR, I help support various research groups utilizing CCR's infrastructure. The research groups come from all disciplines and many of the projects involve writing software to process and analyze vast amounts of scientific data. Typically we build large data warehouses and create applications to allow researchers to do searching, reporting, and updating. I also provide computational support for UB's Next-Generation Sequencing and Expression Analysis Core, which houses an Illumina HiSeq 2000.

Previously, I worked as a Software Engineer for O'Reilly Media developing web applications and data visualization tools. I spent a considerable amount of time working with XML technologies and hacking on O'Reilly's corpus of digital content.

A while back I contributed code to Mark Logic's developer network and wrote cqsh, an XQuery command line shell and mlxcc, a C API enabling access to Mark Logic from Perl, Ruby, PHP, and Python. I was invited to speak at the 2nd (2005) and 3rd (2006) annual Mark Logic User conferences (before in was Mark Logic World) in which I co-presented cqsh, mlxcc, and projects from O'Reilly Labs.

A long time ago I contributed some perl code to Word Hacks, despite my loathing for all software originating from Redmond.

You can also find some of my projects on GitHub.

System Adminstrator

As a system administrator at CCR, I write code to automate software installations across thousands of linux compute nodes using cobbler and puppet. I administer user account management systems using Kerberos and LDAP and maintain core network services such as DNS, DHCP, and NTP. I also build system monitoring tools for tracking resource utilization and energy consumption on high perfomance compute systems. In addition, I adminster web application environments running Apache, Tomcat, and MySQL, and virtual machine infrastructure using KVM and libvirt.

Web Developer

I'm a full stack web developer, i.e., give me vim and a linux box and I'll cook you up a full blown web application faster that you can finish a ziggy piggy.

I lead the development of UBMoD, an open source data warehouse and web portal for mining statistical data from resource managers such as TORQUE.

I built the web infrastructure for miRdSNP, a database exploring disease-associated SNPs and microRNA target sites on 3' untranslated regions of human genes.

While working at O'Reilly Media, I created the web front end for the O'Reilly Research Portal, a system for visualizing book sales and technology trends using Treemaps and spearheaded the development of a content repository for remixing O'Reilly's digital content using Mark Logic and the Atom Publishing Protocol.

In my spare time, I administer an ecommerce site for the Vermont based outdoor retailer KLmountainshop, which sees on average ~45K unique visitors and ~200K page views per month. In 2010, KLmountainshop grew their online presence 67% making Internet Retailers Second 500 list.

Bioinformatician in Training

I'm currently a student in the Department of Biostatistics at the University at Buffalo and Roswell Park Cancer Institue. I'm pursing a Masters in Bioinformatics and Biometry under the guidence of Dr. Song Liu. Currently my research is focusing on algorithms for detecting fusion genes from next-generation sequencing data.

I recently worked on a project which explored the need for biostatisticians to perform correlation analysis on data from array comparative genome hybridization (aCGH). The paper focused on the data-intensive computational problem of processing large correlation matrices on the order of 1 million rows by 1 million columns. I implemented an approach using Hadoop which scaled to 128 compute nodes and queried the top 200 values in under 40 minutes.
